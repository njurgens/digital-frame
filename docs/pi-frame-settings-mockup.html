Navigation: Left sidebar with four sections — Slideshow, Display, Wi-Fi, System. Back button returns to slideshow. No nested navigation; all settings fit one level deep.

Slideshow: Fixed interval segmented control (easier touch targets than free slider). Sync status shown inline with manual trigger button.
Display: Sleep schedule times are grayed out / non-interactive until the schedule toggle is enabled.
Wi-Fi: Connected network pinned at top with destructive Forget. Available networks listed below with signal-strength icons.
System: Shutdown and Reboot are destructive-styled (red). Both require a confirmation dialog before executing.
